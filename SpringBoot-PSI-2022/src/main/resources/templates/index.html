PSI 資料表關聯圖

功能介紹 :

Department : (add/update/delete(有員工時不可刪除))

Employee    : (add/update/delete(員工有工作時無法刪除))

Supplier       : (add/update/delete(供應商有被採購時無法刪除))

Purchase     : (add/update/delete(有細目無法刪除)/細目連結)

PurchaseItem : (add/update/delete)

Cutomer       : (add/update/delete(有訂單時無反刪除))

Order            : (add/update/delete(有細目無法刪除)/細目連結)

OrderItem     : (add/update/delete)

商品操作 : Product(名稱/成本/銷售)

客戶操作 : Customer(名稱/電話)

供應操作 : Supplier(名稱)

部門操作 : Department(名稱/主管) -> Department(員工名稱 / 部門)

訂單操作 : Order(員工/客戶) -> OrderItem(商品/購買數量)

採購操作 : Purchase(員工/供應者) -> PurchaseItem(商品/購買數量)
